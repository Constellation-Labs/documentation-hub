# Introduction

This guide is specifically for setting up your [VPS](/validate/validator/vps) to build a Constellation Network node. It is a general guide.

## ◽ Before We Begin

This guide does not include cloud [provider-specific steps or images](/validate/setup-guides/). 

You should used this guide to understand what needs to be accomplished and adapt it to your specific cloud provider’s control panel. The guide offers opinionated advice with sensible defaults.

## ◽ Create SSH Keys

Details can be found [here](/validate/validator/ssh-keys).

Please choose your platform:

<Tabs style={{backgroundColor:'red'}}>
<TabItem value="Windows" label="Windows">

#### ◽ Open a Command Prompt

From the Microsoft Windows start menu type in `Command Prompt` on the main search bar and select the `Command Prompt` icon.

<WinCommandPrompt>
<p>C:\></p>
</WinCommandPrompt>

#### ◽ Create your SSH key pair

Change to the `.ssh` directory to simplify creation and future locating of your SSH key pairs.
```
cd .ssh
```
Generatate SSH key.
```
ssh-keygen -b 4096 -C ConstellationNetwork
```

<WinCommandPrompt>
C:\Users\myuser\.ssh>ssh-keygen -b 4096 -C ConstellationNetwork<br />
Generating public/private rsa key pair.<br />
Enter file in which to save the key (C:\Users\myuser/.ssh/id_rsa):<br />
</WinCommandPrompt>

Choose a `name` for your key pair (*use whatever name you would like.*)
```
ConstellationNodeKeyPair
```
Enter a complex strong passphrase and confirm the passphrase.
